# 모두의툴 — 유틸 툴 웹사이트 광고수익 전략 + 구축 완료 보고 (2026-07-21)

> 대표 지시: "PDF 변환·합치기·나누기, 이미지 압축, 삼각함수·만나이 계산기 같은 걸 모아 광고수익 웹을 구글에 게시해 돈 버는 전략을 짜라."
> 본 보고 = ① 전략(웹리서치 24에이전트: 5트랙 조사 + 적대적 검증 + 3인 심사) ② **실제 구축 완료된 사이트**(13개 툴, 전부 실동작 검증) ③ 대표 실행 체크리스트.

## 0. 정직한 요약 먼저

- 이 사업은 **72시간 내 수익이 나는 구조가 아니다.** 애드센스 승인 1~2개월 + 구글 신규 사이트 유입 지연 3~6개월이 상수다. **첫 6개월 KPI는 수익이 아니라 색인·랜딩 수**이고, 12개월차 기본 시나리오 월 10만~30만원, 낙관 시나리오 상한이 월 100만원 초반이다(근거는 아래 수익 시나리오).
- 대신 **비용이 거의 0**이다: 서버 없음(전부 브라우저 클라이언트 처리), 연 비용은 도메인 1~2만원뿐. 방치해도 죽지 않는 자산형 사업.
- **개발은 오늘 이미 끝났다.** 전략만 짜지 않고 사이트를 완성했다 — `tools-web/` 폴더, 13개 툴 전부 Playwright 실동작 검증 통과. 대표가 할 일은 도메인 구매(1~2만원)와 계정 신청뿐이다.

## 1. 오늘 구축 완료된 것 (tools-web/)

**13개 툴**: PDF 합치기·나누기·**페이지 삭제/순서변경/회전**·PDF→JPG·이미지→PDF / 이미지 압축(**목표 KB 지정**: 500/300/200/100/50KB)·크기조절/포맷변환 / 만나이·퍼센트·글자수·D-day·삼각함수·평수 계산기.

- 100% 클라이언트 사이드(pdf-lib·pdf.js·browser-image-compression·canvas, 전부 로컬 벤더) → "파일이 서버로 전송되지 않습니다"가 사실이자 차별점. 서버비 0원.
- 페이지마다: SEO 메타+JSON-LD(WebApplication·FAQPage), 애드센스 통과용 한국어 가이드 500단어+ 및 FAQ 5개, 관련도구 내부링크, 광고 슬롯 2개(승인 전엔 DOM 공간 미점유 → CLS 안전, `assets/ads-config.js`에 ID만 넣으면 전 페이지 일괄 활성화).
- 지원 페이지: 허브·소개·개인정보처리방침(광고 쿠키 고지 포함)·이용약관·404. 빌드 스크립트가 sitemap.xml·robots.txt 자동 생성.
- **검증**: ①구조 계약 테스트 전 페이지 통과 ②툴별 QA 에이전트가 실제 파일로 엔드투엔드 실측 — 병합/분할/편집 결과 PDF의 페이지수·회전각, 압축/리사이즈 결과의 매직넘버·해상도 바이트 파싱, 계산기 수치 수학 검증(만나이 윤년·생일당일 엣지 포함). 발견 버그 1건(pdf.js 신 API가 Chromium≤141에서 깨짐)은 폴리필로 수정 완료.
- 다크모드, 모바일 우선, 외부 CDN 0(속도·CWV 유리).

## 2. 리서치가 바꾼 판단 (요약 — 전문은 부록)

1. **1순위 툴은 "PDF 페이지 삭제·순서변경"** (3심사 254점 만장일치 1위): 경쟁 SERP가 가장 무름 — 상위가 로그인 요구(Adobe)·일 2작업 제한(Smallpdf)·블로그 글. 오늘 추가 구축 완료. 향후 "PDF 회전"·"PDF 페이지 추출"을 별도 랜딩으로 쪼개면 1구현=4~5키워드.
2. **이미지 압축은 "목표 KB" 시나리오가 승부처**: "토익 500KB"·"이력서 100KB" 같은 제출 시나리오 랜딩군이 한국어로 사실상 무주공산. 오늘 목표 KB 기능 반영 완료 — 다음 스프린트에서 시나리오별 랜딩 20~30장 확장.
3. **만나이·연봉·시급·BMI는 유입 기대 낮음**(네이버 위젯+대형 포털 점령). 이미 만든 만나이·삼각함수는 콘텐츠 질량·내부링크용으로 유지하되 광고비 쓸 대상 아님.
4. **"국내 최초 무업로드" 카피 금지** — 폴라리스오피스툴즈·imagesize 등이 동일 콘셉트 선점(검증에서 확인). 승부는 콘셉트가 아니라 **롱테일 랜딩 선점 속도 + 콘텐츠 결합**.
5. FAQ 리치결과는 2026-05 종료됨 — FAQ 마크업은 AI 검색(AI Overviews) 인용 대비용으로 유지. **가짜 평점(aggregateRating) 삽입 절대 금지**(스팸 수동조치 위험).
6. 애드센스는 **무료 서브도메인(netlify.app 등) 사실상 승인 불가** → 커스텀 도메인 필수.

## 3. 대표 실행 체크리스트 (이것만 하면 됨)

**이번 주 (총 1~2만원, 30분)**
1. [ ] **도메인 구매** — 후보: modutool.com / modu.tools / modutool.kr (등록 가능 여부는 구매 시점 확인). 구매 후 알려주면 배포까지 내가 마무리.
2. [ ] 배포 계정 선택 — 기존 Cloudflare Pages(어닝옵스와 동일 방식) 권장, Netlify도 무방. `node scripts/build.mjs https://도메인` → `dist/` 게시가 전부.
3. [ ] Google Search Console + 네이버 서치어드바이저 + 다음 등록(도메인 소유 확인 각 5분, 절차는 부록 ④).

**오픈 직후**
4. [ ] **카카오 애드핏 신청**(심사 1~2영업일, 승인 공백기 수익원) — 스크립트는 이미 사이트에 준비됨, 발급받은 DAN 코드만 `assets/ads-config.js`에 기입.
5. [ ] 문의용 이메일 주소 결정(contact@도메인 등) → about 페이지에 기입.

**1~2개월차**
6. [ ] **구글 애드센스 신청** — 신청 전 블로그성 가이드 글 8~15개 보강 필요(내가 작성 가능). 거절-보강-재신청 루프는 정상 과정.
7. [ ] (애드센스 승인 후) 쿠팡 파트너스는 맥락 맞는 페이지에만 선별 — 우선순위 낮음.

**착수 전 확인 권장**: 네이버 검색광고 키워드도구/블랙키위에서 상위 10개 키워드 검색량 실측(리서치 환경에선 로그인 장벽으로 미확보 — 최대 불확실성). 국내 실제 SERP도 로컬 브라우저로 재확인.

## ② 검증에서 반박된 주장과 정정

**완전 반박 (전략 수정 반영)**

1. **"폴라리스는 뷰어만 무업로드, 클라이언트 사이드 HWP 변환은 난이도로 사실상 부재" → 반박됨.** 폴라리스오피스 공식 "Polaris Convert HWP to PDF"(Chrome 확장·Android 앱)가 로컬 변환을 무료 무제한 제공하고, 독립 확장 "HWP to PDF", 오픈소스 hwp.js·rhwp(WASM)도 존재. → 정정: HWP 니치는 실재하나 클라이언트 사이드 변환은 이미 경쟁 제품이 있는 영역. **"국내 최초/유일 무업로드" 류 카피는 어떤 툴에도 쓰지 않는다**(폴라리스·iTool·imagesize가 이미 콘셉트 선점). HWP 변환은 완전한 레이아웃 충실도 부담이 여전히 커서 6개월 로드맵에서 제외 유지 — 단 사유는 "불가능"이 아니라 "투자 대비 효율".
2. **"쿠팡 광고 썸네일 가이드 강화는 2026년 5월부터" → 반박됨.** 실제는 **2025년 5월 시행**(이미 1년 이상 경과 — 신선한 수요 촉발 이벤트가 아님). 자동 누끼·리사이즈도 "위반 시 자동"이 아니라 광고센터 셀러 동의 시 적용이라는 보도 있음. "마켓별 프리셋 도구 전무"도 부분 반박(pickjpg 등 존재). → 정정: 셀러 썸네일 도구는 "전 채널 프리셋 특화가 드물다" 수준의 기회로 하향 조정, 2차 로드맵으로 이연.
3. **"imagesize.net이 헤드 키워드에 노출 중 = 무업로드 포지셔닝 SEO 유효성 검증" → 과장으로 판정.** 미국 구글 기준 '사진 용량 줄이기' 등 상위 10위 내 미노출, 색인만 확인. → 정정: 무업로드 포지셔닝이 SEO에서 통한다는 실증은 아직 없다. 본 전략의 승부처는 포지셔닝 자체가 아니라 **롱테일 랜딩 선점 + 콘텐츠 결합**이다.

**표현 정정 (사실관계 보정, 방향은 유지)**

4. Smallpdf 무료 제한은 "하루 문서 2개"가 아니라 **"하루 작업(task) 2회"**. Adobe는 업로드까지는 무로그인, **편집 진행·저장 시 로그인 요구**.
5. 알PDF "누적 1,500만 설치"는 **2020년 10월 기준**(약 5년 8개월 경과), "61% 직장인"은 누적 설치자가 아니라 **당시 MAU 214만 명 중 비율**(닐슨 코리안클릭). 수요 대리 지표로만 사용하고 시점 명시.
6. 유틸 RPM 추정 하단 확대: "1,000~3,000원"이 아니라 **500~3,000원(1,000원 미만 가능)**. 일 1만 PV 시 월 30만~90만원 → **15만~90만원**이 보수적 범위 (⑤에 반영).
7. 네이버 이자계산기 위젯에 **중도상환수수료 탭이 이미 존재** — 대출 계산기의 차별화 폭은 '시나리오 비교'에 한정. 맞춤법도 네이버 별도 위젯(500자)이 있어 단독 차별점 아님.
8. "최신 요율 반영만으로 차별화" → 과장. 대형 포털·롱테일 사이트 전원이 '2026 반영'을 내걸고 있어 **최신성은 차별화가 아니라 기본 요건**.
9. "장문 콘텐츠는 애드센스 필수 요건" → 구글 명시 요건이 아닌 **승인 확률을 크게 높이는 사실상의 관행**. '콘텐츠 부족 최빈 거절 사유'도 공식 통계 없는 일화적 주장.
10. 구글 샌드박스는 **구글이 공식 부인**(존 뮬러 2018·2021) — 실무 관측치(3~6개월, 최대 9개월)로만 취급.

## ③ 수익화 로드맵: 애드핏 → 애드센스 → 쿠팡파트너스

**0단계 — 선결 조건 (사이트 오픈 전)**
- [ ] **커스텀 도메인 구매(연 1~2만원)**: *.netlify.app / *.vercel.app 등 무료 서브도메인은 애드센스 사실상 승인 불가(실증 사례 다수). 처음부터 커스텀 도메인+HTTPS로 운영
- [ ] 필수 3페이지(소개·개인정보처리방침·문의) 뼈대 단계 구현
- [ ] 대표 도메인 통일(www/비www 중 하나, 301 리다이렉트)

**1단계 — 카카오 애드핏 (오픈 직후, 승인 공백기 수익원)**
- [ ] 광고 스크립트 **먼저 설치**(광고 요청 정상 호출 상태가 심사 전제 — 미설치가 보류 사유 1위) → 매체 등록 신청
- [ ] 심사 영업일 1~2일 기본(지연 사례 있음). 일반 웹사이트에 "개설 30일+글 20개" 조건이 적용되는지는 공식 미확인 — 거절 시 콘텐츠 보강 후 재신청
- [ ] 기대치: CPC 과금, 체감 수익은 애드센스의 20~30% 수준(일화적 수치, 편차 큼). 목적은 수익보다 "첫 수익 발생 시점 앞당기기"

**2단계 — 구글 애드센스 (본 수익원, 오픈 후 1~2개월 차 신청)**
- [ ] 신청 시점 기준: 완성된 툴 페이지 10개+ (각 페이지 ⑥의 콘텐츠 템플릿 충족), 블로그성 가이드 글 8~15개
- [ ] 미완성·씬 페이지는 noindex 처리 후 신청
- [ ] 심사 소요: 공식 며칠~2주, 실후기 3일~14일 최빈, 1개월+ 사례 있음. 'Low value content' 거절 시 콘텐츠 보강 후 재신청 (거절-재신청 루프를 일정에 미리 반영)
- [ ] 승인 후 애드핏과 병행 가능(정책 충돌 없음, 광고 위치만 겹치지 않게)
- [ ] 광고 배치: 고정 슬롯 1~2개 절제 운영 — CWV(특히 CLS·INP) 보호 + '유용한 사이트 모음' 자연 백링크 유도에 유리

**3단계 — 쿠팡 파트너스 (애드센스 승인 후에만)**
- [ ] 승인 심사 중 제휴 링크는 '콘텐츠 대비 광고 과다' 판정에 불리할 수 있어 **애드센스 승인 후 삽입**이 안전
- [ ] 상품 맥락이 자연스러운 툴에만 선별 삽입(수수료 카테고리별 최대 1~3% 수준·수시 변경, 가입 후 공식 요율 확인)
- [ ] 대가성 문구("이 포스팅은 쿠팡 파트너스 활동의 일환으로...")를 링크 있는 전 페이지 공통 컴포넌트로 구현
- [ ] 누적 판매 15만원 도달 시 최종 채널 심사(활동 페이지 URL·문구 스크린샷) — 전월 실적 1만원 이하는 정산 이월
- 유틸 사이트 특성상 우선순위 낮음: 맥락 맞는 툴이 제한적(예: 평수 계산기→인테리어 자재 정도). 보조 수익원으로만

**배제**: Mediavine·Ezoic 등 프리미엄 애드네트워크는 Tier1(영어권) 트래픽 50%+ 요건으로 한국어 사이트에 구조적 무관. 장기 옵션으로 영어판 툴 페이지 병설만 검토 가치.

## ⑤ 현실적 수익 시나리오

**공통 전제(정직한 한계)**: 유틸 사이트 전용 한국 RPM 공개 실측치는 존재하지 않음. 아래는 블로그 RPM 자료(통상 1,500~10,000원, 국가 벤치마크 ~4,500원, 카테고리 하단 300원~) 기반 추정으로 검증 트랙이 보정한 범위 **500~3,000원/1,000PV**를 사용. 애드핏 병행 +20~30%(일화적). 실제 ±50% 이상 변동 가능.

| 구간 | 보수 | 기본 | 낙관 |
|---|---|---|---|
| 1~6개월차 | **월 0~2만원** | 월 0~5만원 | 월 0~10만원 |
| 7~12개월차 | **월 2만~10만원** | **월 10만~30만원** | 월 30만~60만원 |
| 12~18개월차 | 월 5만~15만원 | 월 15만~40만원 | **월 45만~100만원** |

- **1~6개월차가 0원대인 근거**: 샌드박스성 유입 지연 3~6개월 + 애드센스 승인 자체가 2~3개월차. 이 기간 수익은 애드핏 소액+메이커 플랫폼 직접 유입분뿐. **초기 6개월은 수익이 아니라 색인·랜딩 축적이 KPI**.
- **보수** (확률 높음): 롱테일 일부만 진입, 일 500~2,000 PV(월 1.5만~6만 PV) × RPM 700~1,500원. 헤드 전멸 가정.
- **기본**: 시나리오 랜딩 30~50장 중 상당수 진입 + 계산기류 안착, 일 3,000~5,000 PV × RPM 1,000~2,000원 + 애드핏 20~30%.
- **낙관**: PDF 헤드 1~2개 상위 진입 + 목표 KB 랜딩군 성공, 일 1만 PV × RPM 1,500~3,000원 + 애드핏·쿠팡. **일 1만 PV 도달 자체가 12개월+ 과제**라는 점을 명시 — 참고로 이슈 블로그 기준 월 100만원엔 일 방문 2,000~3,000명+고CPC가 필요하다는 후기가 있고, 유틸은 CPC가 그보다 낮다.
- 수익 상향 레버: 부동산·대출 인접 페이지(평수, 예비 후보인 중도상환 시뮬레이터)는 한국 애드센스 최고 단가 카테고리(금융 RPM 10,000~30,000원 주장 존재 — 단 블로그 경험담 기반) — 동일 트래픽 대비 수 배 차이 가능하나 확정 수치 아님.

## ⑥ 페이지 콘텐츠 요건 (애드센스 통과용)

거절 사유 1위 'Low value content'의 전형이 "입력폼+SEO 문단 2~3줄"인 툴 페이지다. 통과 사례의 공통 패턴을 표준 템플릿으로 강제한다.

**툴 페이지 표준 템플릿 (전 페이지 공통, 본문 1,500~2,000자+)**
1. 툴 UI 최상단 (즉시 사용 가능)
2. "서버 업로드 없음" 신뢰 블록: 처리 방식 1~2문장 + 개인정보처리방침 링크
3. 사용법 (스크린샷 또는 단계별 3~5스텝)
4. 작동 원리·개념 설명 (계산기는 정의+계산식+법적 근거 — 상위 사이트들 스니펫이 전부 이 설명글에서 추출됨 = 랭킹 자산 겸용)
5. **실제 예시(worked example)**: 계산기는 구체 수치 예시, 파일 툴은 상황 예시("토익 접수용 500KB 만들기")
6. FAQ 4~6개 (가시 텍스트 + FAQPage 마크업 일치 — 리치결과 아닌 AI 인용 대비용)
7. 관련 툴 내부링크 4~6개
8. 갱신일·연도 표기 (계산기는 "2026년 기준" 배지)

**사이트 레벨**
- 필수 3페이지: 소개(운영 목적·처리 방식), 개인정보처리방침(클라이언트 처리 명시 — 이 사이트의 경우 실제로 강력한 차별 문구가 됨), 문의(이메일 폼)
- 블로그 섹션 8~15개 글: 툴 튜토리얼·비교("PDF 합치기 사이트 5곳 비교")·보안 가이드("온라인 PDF 사이트에 회사 문서 올려도 되나" — 수요 실증된 주제)
- AI 양산 티 나는 균질 문체 금지(2025년 이후 원본성·E-E-A-T 심사 강화), 전 페이지 HTTPS
- 신청 시점에 빈 페이지·"준비 중" 페이지 0개 (미완성은 noindex)
- 심사 중 광고·제휴 링크 최소화 (쿠팡 링크는 승인 후)

### 한 줄 결론
**"무업로드 한국어 툴"이라는 콘셉트는 이미 선점됐지만 SERP는 아무도 못 잡았다.** 승부는 콘셉트가 아니라 (1) 심사 전원이 1위로 꼽은 PDF 조작계 롱테일과 목표 KB 압축 시나리오 랜딩의 선점 속도, (2) 애드센스 통과 가능한 콘텐츠 템플릿의 일관 적용, (3) 샌드박스 6개월을 버티는 비구글 유입 확보에 있다. 첫 6개월 수익 기대치는 0원대가 정직한 숫자이며, 12개월차 기본 시나리오 월 10만~30만원, 낙관해도 월 100만원 초반이 상한이다 — 착수 전 키워드 실측으로 이 가정부터 검증하라.
