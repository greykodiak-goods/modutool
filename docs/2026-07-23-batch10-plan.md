## 카테고리 구조 변경
- **util 카테고리 신설**: 허브에 "텍스트·생활 유틸" 섹션 추가(id="util"). CAT_SLUGS.util = [qr-generator, password-generator, text-diff, char-count(기존 이동 아님 — calc 유지, 중복 배치 안 함)].
- **사이트 매핑**: util은 당분간 **calc 사이트(ThisIsMyCalculator)에 포함**(sites.json calc.categories = ["calc","util"]) — 계산기·유틸 묶음이 자연스럽고 사이트 난립 방지. 수요가 커지면 분리.
- 이미지 4종은 image 카테고리 → ThisIsMyIMG(/img/)에 자동 포함.
