PDF 페이지 추출🔒 서버 업로드 없음

긴 PDF에서 필요한 쪽만 골라 새 PDF 한 개로 뽑아냅니다. 설치·가입·용량제한 없이 무료.

📑 페이지를 추출할 PDF 파일을 여기에 끌어다 놓거나 클릭해서 선택
파일 1개 — 선택하면 총 페이지 수가 바로 표시됩니다

    사용 방법

    1. 위 영역에 PDF 파일 1개를 끌어다 놓거나, 클릭해서 선택합니다. 파일을 읽는 즉시 총 페이지 수가 목록에 표시됩니다.
    2. 추출할 페이지를 1-3,5,7-9처럼 쉼표와 하이픈으로 입력합니다. 처음에는 전체 범위(1-마지막쪽)가 채워져 있으니 필요한 부분만 남기고 고치면 됩니다.
    3. 페이지 추출 버튼을 누르면 즉시 처리되어 원본명_추출.pdf 파일이 자동으로 다운로드됩니다. 원본 파일은 그대로 남습니다.

    이 도구가 다른 페이지 추출 사이트와 다른 점

    흔한 온라인 PDF 도구는 파일을 자기 회사 서버에 업로드한 뒤 페이지를 뽑아서 되돌려줍니다. 업로드된 문서가 언제 삭제되는지, 처리 과정에서 누가 열람할 수 있는지 사용자는 확인할 방법이 없습니다. 계약서, 주민등록등본, 진단서, 급여명세서처럼 개인정보가 담긴 문서라면 더욱 불안할 수밖에 없습니다.

    모두의툴의 PDF 페이지 추출은 파일을 어디로도 보내지 않습니다. 브라우저에 내장된 기술로 페이지 추출 작업 전체가 내 기기 안에서만 이뤄집니다. 그래서 세 가지 장점이 생깁니다.

    • 보안 — 파일이 기기 밖으로 나가지 않으므로 유출 가능성 자체가 없습니다.
    • 속도 — 업로드·다운로드 대기가 없어 수십 MB 문서도 몇 초 안에 끝납니다.
    • 무제한 — 서버 비용이 들지 않으니 파일 용량·횟수를 제한하지 않습니다.

    페이지 범위 입력 방법

    페이지 번호를 쉼표로 구분해 입력하면 해당 페이지만 모은 새 PDF 한 개가 만들어집니다. 연속 구간은 하이픈으로 줄여 쓸 수 있고, 입력한 순서 그대로 새 PDF에 배치됩니다.

    • 5 — 5쪽 한 장만 추출
    • 2-4 — 2쪽부터 4쪽까지 연속 추출
    • 1-3,5,7-9 — 1~3쪽 + 5쪽 + 7~9쪽, 총 7쪽을 입력한 순서대로 배치

    입력한 범위가 실제 페이지 수를 벗어나거나(예: 30쪽짜리 문서에 1-40 입력), 시작 쪽이 끝 쪽보다 크거나, 숫자·쉼표·하이픈이 아닌 문자가 섞이면 어떤 부분이 문제인지 바로 알려주므로, 페이지 수가 많은 문서도 실수 없이 뽑을 수 있습니다. 같은 쪽을 여러 번 적으면 그만큼 반복해서 들어가므로 표지를 두 번 넣는 것도 가능합니다. 반대로 모든 페이지를 낱장 PDF 여러 개로 나누고 싶다면 이 도구 대신 PDF 나누기의 전체 분할 모드를 사용하세요 — 페이지마다 개별 파일이 만들어집니다.

    이럴 때 쓰세요

    • 수십 쪽짜리 계약서에서 서명 페이지나 특약 부분만 뽑아 상대방·기관에 제출할 때
    • 논문·보고서에서 특정 챕터나 초록만 추려 지도교수·동료에게 공유할 때
    • 한 번에 스캔한 묶음 파일에서 등본·통장사본·신분증 사본이 있는 쪽만 분리해 보관할 때
    • 전자책·강의자료에서 필요한 단원만 뽑아 태블릿에 넣거나 인쇄할 때
    • 발표자료에서 배포용 요약본(표지+핵심 페이지)만 추려 만들 때
    • 이메일 용량 제한에 걸린 대용량 PDF에서 꼭 필요한 페이지만 추려 가볍게 보낼 때
    💡 팁: 뽑은 페이지의 순서를 바꾸거나 회전·삭제까지 하고 싶다면 PDF 페이지 정리를, 추출한 PDF를 다른 문서와 하나로 묶으려면 PDF 합치기를 이어서 사용하세요. 세 도구 모두 서버 업로드 없이 같은 방식으로 동작합니다.

    자주 묻는 질문

    파일이 정말 서버로 전송되지 않나요?

    네. 추출 작업 전체가 브라우저에 내장된 기술로 내 컴퓨터(또는 휴대폰) 안에서만 처리됩니다. 개발자도구의 네트워크 탭을 열어 확인해도 파일 업로드 요청이 없습니다. 계약서·등본·진단서 같은 민감한 문서도 안심하고 사용할 수 있습니다.

    페이지 범위는 어떻게 입력하나요?

    쉼표로 구분하고 연속 구간은 하이픈으로 적습니다. 예를 들어 1-3,5,7-9를 입력하면 1~3쪽, 5쪽, 7~9쪽 — 총 7쪽이 입력한 순서대로 새 PDF 한 개로 만들어집니다. 한 쪽만 뽑으려면 숫자 하나(예: 5)만 적으면 됩니다.

    추출한 파일의 이름은 어떻게 되나요?

    원본 파일명 뒤에 _추출이 붙습니다. 예를 들어 계약서.pdf에서 페이지를 뽑으면 계약서_추출.pdf로 다운로드됩니다. 원본 파일은 전혀 변경되지 않고 그대로 남습니다.

    암호가 걸린 PDF에서도 페이지를 추출할 수 있나요?

    열람 암호(비밀번호)가 걸린 PDF는 보안 정책상 열 수 없어 안내 메시지가 표시됩니다. PDF 뷰어 등에서 암호를 해제해 저장한 파일로 다시 시도해 주세요.

    추출한 PDF의 화질이나 내용이 달라지나요?

    아니요. 페이지를 다시 렌더링하지 않고 원본 페이지 데이터를 그대로 복사하는 방식이라 글자·이미지 화질, 글꼴, 링크가 원본과 동일하게 유지됩니다.
