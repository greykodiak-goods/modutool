How to Use

  1. Drop the screenshot or photo into the area above, or click to select it.
  2. Pick a masking style — Mosaic pixelates the area, Black-out covers it completely. You can change the style between boxes and mix both in one image.
  3. Drag across the preview to cover each sensitive area: account numbers, names, phone numbers, faces, license plates. Every drag adds another box; Undo Last Box removes the most recent one.
  4. Click Download Masked Image. The masked copy downloads as filename_masked, and your original file on disk is untouched.

Why masking sensitive info in the browser matters

The images people redact are, almost by definition, the most sensitive files they own — bank-transfer screenshots, ID cards, contracts, chat logs. Uploading exactly those images to an unknown server to "hide private info" is backwards. This tool draws and masks entirely inside your browser: the image never leaves your device, and closing the tab destroys everything.

A second benefit of how saving works: the output file is rebuilt from a clean canvas, so hidden EXIF metadata — GPS location, capture time, camera model — is not carried over into the masked copy.

Mosaic or black-out — which should you use?

  • Mosaic (pixelate) — natural-looking, good for faces, backgrounds, and general context you want softened. Choose Strong for text you want unreadable.
  • Black-out — removes the information entirely. Use it for critical identifiers: resident registration numbers, account numbers, addresses, signatures. A black box can never be "enhanced" back.
💡 Tip: For text that must stay secret, prefer Black-out. Research has shown weak pixelation over known fonts can sometimes be reconstructed — solid boxes cannot.

Frequently Asked Questions

Is my screenshot really never uploaded to a server?

Correct. Blurring and blacking out happen entirely on your device using the canvas feature built into your browser. If you open the Network tab of your browser's developer tools, you will see no file upload request. That is exactly why this tool is safe for bank screenshots, ID photos, contracts, and chat captures.

Can a mosaic (pixelation) be reversed to reveal the original?

The saved file only contains the averaged color blocks — the original pixels are not stored anywhere in it, so it cannot simply be "decoded". However, very weak pixelation over large text can sometimes be guessed by AI tools. For truly critical information such as resident registration numbers or account numbers, use the solid black-out mode, which removes the information completely.

Can I hide several areas in one image?

Yes. Draw as many boxes as you need — each drag adds another masked region, and you can mix pixelated boxes and black boxes in the same image. Made a mistake? Undo removes only the last box, and Clear areas starts the masking over without reloading the image.

Does the tool change the image quality or size?

Everything outside the boxes you draw keeps its original pixels. The file is re-encoded on save — JPG stays JPG and PNG or WEBP is saved as PNG with transparency preserved — so the visible quality is unchanged while the masked regions are permanently replaced.

Does it also remove hidden location (EXIF) data?

Yes, as a side effect of saving: the output is generated from a clean canvas, so EXIF metadata including GPS position, camera model, and capture time is not carried over. If you only want to strip metadata without masking anything, use our dedicated EXIF removal tool.
