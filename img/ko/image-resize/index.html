이미지 크기조절·변환🔒 서버 업로드 없음

사진 여러 장의 해상도를 픽셀·퍼센트로 조절하고 JPG·PNG·WEBP로 변환합니다. 설치·가입·용량제한 없이 무료.

🖼️ 이미지를 여기에 끌어다 놓거나 클릭해서 선택
여러 장을 한 번에 선택할 수 있습니다 (JPG·PNG·WEBP·GIF·BMP)

    사용 방법

    1. 위 영역에 이미지를 끌어다 놓거나 클릭해서 선택합니다. JPG·PNG·WEBP·GIF·BMP를 지원하며 여러 장을 한 번에 올릴 수 있습니다.
    2. 크기를 정합니다 — 가로 픽셀을 직접 입력하거나(비율 고정이 기본이라 세로는 자동 계산됩니다), 원본의 75%·50% 크기로 줄이는 방식 중 고릅니다.
    3. 포맷(JPG·PNG·WEBP)과 품질을 정합니다. 품질 슬라이더는 JPG·WEBP에만 적용되며, 웹 업로드용이라면 80~90이 화질과 용량의 균형점입니다.
    4. 크기 조절하기 버튼을 누르면 파일마다 변환 전/후 해상도와 용량이 표시되고, 한 장씩 또는 전체를 한 번에 다운로드할 수 있습니다.

    이 도구가 다른 이미지 크기 조절 사이트와 다른 점

    대부분의 온라인 이미지 도구는 사진을 자기 회사 서버에 업로드한 뒤 처리해서 되돌려줍니다. 업로드된 사진이 언제 지워지는지, 처리 과정에서 누가 볼 수 있는지 사용자는 알 수 없습니다. 아이 사진, 신분증 사본, 계약 관련 스캔본처럼 민감한 이미지라면 불안할 수밖에 없습니다.

    ThisIsMyIMG의 이미지 크기 조절은 파일을 어디로도 보내지 않습니다. 브라우저에 내장된 캔버스 기술로 리사이즈·변환 전체가 내 컴퓨터(또는 휴대폰) 안에서만 이뤄집니다. 그래서 세 가지 장점이 생깁니다.

    • 보안 — 사진이 기기 밖으로 나가지 않으므로 유출 가능성 자체가 없습니다.
    • 속도 — 업로드·다운로드 대기가 없어 수십 장도 몇 초 안에 끝납니다.
    • 무제한 — 서버 비용이 들지 않으니 장수·횟수·용량을 제한하지 않습니다.

    화질 처리 방식도 다릅니다. 4,000px짜리 사진을 한 번에 800px로 줄이면 브라우저 기본 리샘플링의 한계 때문에 가는 선이 지글거리는 계단 현상이 생기기 쉽습니다. 이 도구는 절반씩 여러 단계로 나눠 축소한 뒤 마지막에 목표 크기로 맞추는 단계적 축소 방식을 써서 큰 폭으로 줄여도 문자·머리카락·격자무늬 같은 세밀한 부분의 선명도를 지킵니다. 또 스마트폰으로 세로로 찍은 사진의 EXIF 회전 정보를 자동으로 반영하므로, 다른 도구에서 흔히 겪는 "변환했더니 사진이 옆으로 누워 나오는" 고전적인 문제도 없습니다.

    포맷 변환은 실무에서 생각보다 자주 필요합니다. 투명 배경이 필요 없는 사진을 PNG로 들고 있으면 용량만 커지므로 JPG나 WEBP로 바꾸는 편이 낫고, 반대로 로고나 아이콘처럼 투명 배경이 필요한 이미지는 PNG 또는 WEBP를 유지해야 합니다. 이 도구 하나로 크기 조절과 포맷 변환을 한 번에 끝낼 수 있어, 여러 사이트를 오가며 사진을 올렸다 내렸다 할 필요가 없습니다.

    이럴 때 쓰세요

    • 쇼핑몰·스마트스토어 상품 이미지를 가로 1000px 같은 픽셀 규격에 맞출 때
    • 블로그·카페·게시판의 첨부 용량 제한에 걸려 사진 크기 줄이기가 필요할 때
    • 홈페이지 로딩 속도를 높이려고 JPG·PNG를 WEBP 변환할 때
    • 이메일·업무 메신저로 보내기 전에 수십 MB 원본 사진을 한꺼번에 축소할 때
    • 이력서·지원서·증명사진처럼 제출처가 픽셀 크기를 지정한 파일을 만들 때
    💡 팁: 해상도(픽셀)는 그대로 두고 용량만 줄이고 싶다면 이미지 압축이 더 알맞습니다. 반대로 제출 규격이 픽셀로 정해져 있다면 이 도구로 크기를 먼저 맞추고, 그래도 용량이 크면 압축 도구를 이어서 쓰면 됩니다.

    자주 묻는 질문

    사진이 정말 서버로 업로드되지 않나요?

    네. 크기 조절과 포맷 변환은 브라우저에 내장된 캔버스 기능으로 내 기기 안에서만 처리됩니다. 개발자도구의 네트워크 탭을 열어 확인해도 파일 업로드 요청이 없습니다. 가족사진이나 신분증 사본 같은 민감한 이미지도 안심하고 사용할 수 있습니다.

    크기를 줄이면 화질이 많이 나빠지지 않나요?

    큰 이미지를 한 번에 확 줄이면 계단 현상이 생기기 쉬운데, 이 도구는 절반씩 여러 단계로 나눠 축소한 뒤 최종 크기로 맞추는 단계적 축소 방식을 사용해 선명도를 최대한 지킵니다. JPG·WEBP는 품질 85 안팎이면 화질 저하를 거의 느끼기 어렵습니다.

    WEBP로 변환하면 무엇이 좋은가요?

    WEBP는 같은 화질에서 JPG보다 보통 25~35% 작게 저장되는 최신 이미지 형식입니다. 블로그·쇼핑몰·홈페이지에 올리면 로딩 속도가 빨라지고, 크롬·엣지·사파리·삼성인터넷 등 모든 최신 브라우저에서 열립니다. PNG처럼 투명 배경도 지원합니다.

    아이폰 HEIC 사진도 변환되나요?

    HEIC는 대부분의 브라우저가 직접 읽지 못해 변환에서 제외될 수 있습니다. 아이폰의 설정 → 카메라 → 포맷에서 '높은 호환성'을 선택해 JPG로 촬영하거나, 사진 앱에서 공유할 때 JPG로 보내면 바로 사용할 수 있습니다.

    세로로 찍은 사진이 옆으로 눕지 않나요?

    스마트폰 사진에 저장된 EXIF 회전 정보를 자동으로 반영해, 변환 후에도 원래 보이던 방향 그대로 저장됩니다. 회전 정보 때문에 사진이 눕거나 뒤집혀 나오는 문제를 걱정하지 않아도 됩니다.
