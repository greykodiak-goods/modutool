이미지 압축🔒 서버 업로드 없음

사진 용량 줄이기 — 화질 조절·해상도 제한·JPG/WEBP 변환을 한 번에. 설치·가입 없이 무료.

🗜️ 이미지 파일을 여기에 끌어다 놓거나 클릭해서 선택
JPG · PNG · WEBP · BMP, 여러 장 한 번에 가능

      사용 방법

      1. 위 영역에 사진(JPG·PNG·WEBP·BMP)을 끌어다 놓거나, 클릭해서 여러 장을 한 번에 선택합니다.
      2. 화질 슬라이더를 조절합니다. 낮출수록 용량이 많이 줄고, 70~80%가 대부분의 용도에 적당합니다.
      3. 필요하면 최대 해상도(긴 변 1920px/2560px)와 출력 형식(JPG/WEBP 변환)을 선택합니다.
      4. 이미지 압축하기 버튼을 누르면 파일별로 처리되며, 목록에 원본 → 압축 용량과 절감률(예: 2.4 MB → 340 KB, -86%)이 표시됩니다.
      5. 파일별 ⬇ 버튼으로 하나씩 받거나, 전체 다운로드로 한 번에 저장합니다. 원본 파일은 그대로 남습니다.

      이 도구가 다른 이미지 압축 사이트와 다른 점

      대부분의 온라인 압축 사이트는 사진을 자기 회사 서버에 업로드한 뒤 처리해서 되돌려줍니다. 업로드된 사진이 언제 지워지는지, 처리 과정에서 누가 볼 수 있는지 사용자는 알 수 없습니다. 가족 사진, 신분증 사본, 계약서 스캔처럼 민감한 이미지라면 찜찜할 수밖에 없습니다.

      모두의툴의 이미지 압축은 사진을 어디로도 보내지 않습니다. 브라우저에 내장된 캔버스 기술로 압축 전체가 내 컴퓨터(또는 휴대폰) 안에서만 이뤄집니다. 그래서 세 가지 장점이 생깁니다.

      • 보안 — 사진이 기기 밖으로 나가지 않으므로 유출 가능성 자체가 없습니다.
      • 속도 — 업로드·다운로드 대기가 없어 수십 장도 몇 초~수십 초면 끝납니다.
      • 무제한 — 서버 비용이 들지 않으니 장수·횟수·용량을 제한하지 않습니다.

      어떤 설정으로 압축해야 하나요?

      이미지 용량은 크게 두 가지가 좌우합니다. 하나는 압축 품질(JPG·WEBP 같은 손실 압축 형식이 색 정보를 얼마나 정밀하게 남기는가), 다른 하나는 해상도(픽셀 수)입니다. 스마트폰 카메라 원본은 보통 4000px가 넘는데, 모니터·휴대폰 화면 표시에는 긴 변 1920px이면 충분해서 해상도만 줄여도 용량이 크게 감소합니다.

      • 웹 게시·커뮤니티 첨부 — 화질 70% + 긴 변 1920px. 화면으로 보기에 차이가 거의 없으면서 절감률이 가장 큽니다.
      • 문서 제출·인쇄 가능성 있음 — 화질 80~90% + 긴 변 2560px 또는 원본 해상도 유지.
      • PNG 스크린샷·그림 — 출력 형식을 WEBP(권장) 또는 JPG로 바꾸면 극적으로 줄어듭니다. 단, 투명 배경이 있는 PNG를 JPG로 바꾸면 투명 영역이 배경색으로 채워지므로 투명도가 필요하면 WEBP를 선택하세요.

      압축 결과가 원본보다 커지는 경우(이미 최적화된 이미지)에는 자동으로 원본을 유지하고 알려드리니, 몇 번이고 부담 없이 시도해 보세요. 파일을 다시 올릴 필요 없이 설정만 바꿔 재압축할 수 있습니다.

      💡 팁: 압축과 함께 개인정보 보호 효과도 있습니다 — 압축 과정에서 촬영 위치(GPS)·기기 정보가 담긴 EXIF 메타데이터가 제거되어, SNS나 중고거래에 사진을 올릴 때 위치 노출 걱정을 덜 수 있습니다. 픽셀 크기를 정확한 수치로 지정하고 싶다면 이미지 크기조절을 이용하세요.

      이럴 때 쓰세요

      • 쇼핑몰·블로그에 올릴 상품 사진이 업로드 용량 제한에 걸릴 때
      • 이메일 첨부 한도(보통 10~25MB)를 넘는 사진 여러 장을 한 번에 첨부해야 할 때
      • 홈페이지 이미지 용량을 줄여 로딩 속도와 검색 노출(SEO)을 개선할 때
      • 입사지원·민원 사이트가 요구하는 "1MB 이하" 같은 첨부 조건을 맞출 때
      • 휴대폰 사진을 정리해 저장공간과 클라우드 용량을 아낄 때

      자주 묻는 질문

      파일이 정말 서버로 전송되지 않나요?

      네. 압축은 브라우저에 내장된 캔버스 기술로 내 기기 안에서만 처리됩니다. 개발자도구의 네트워크 탭을 열어 확인해도 파일 업로드 요청이 없습니다. 신분증 사본이나 가족 사진처럼 민감한 이미지도 안심하고 사용할 수 있습니다.

      화질이 눈에 띄게 나빠지지 않나요?

      화질 80~90% 설정에서는 대부분의 사진에서 육안으로 차이를 구분하기 어렵습니다. 웹 업로드나 문서 첨부 용도라면 화질 70~80%에 긴 변 1920px 설정이 용량과 화질의 균형이 가장 좋습니다. 결과를 바로 내려받아 비교해 보고, 마음에 들지 않으면 설정을 바꿔 다시 압축하면 됩니다.

      어떤 이미지 형식을 지원하나요?

      JPG, PNG, WEBP, BMP를 지원합니다. 아이폰의 HEIC 형식은 브라우저가 직접 읽지 못하므로 사진 앱에서 JPG로 내보낸 뒤 사용해 주세요. PNG는 무손실 형식이라 그대로 압축하면 효과가 작을 수 있는데, 출력 형식을 JPG나 WEBP로 바꾸면 크게 줄어듭니다.

      압축했는데 용량이 줄지 않거나 오히려 커졌어요.

      이미 강하게 압축된 이미지를 다시 압축하면 더 줄어들 여지가 없어 결과가 원본보다 커질 수 있습니다. 이 경우 도구가 자동으로 원본을 그대로 유지하고 알려드립니다. 최대 해상도를 1920px로 제한하거나 WEBP 변환을 선택하면 추가로 줄어드는 경우가 많습니다.

      휴대폰에서도 되나요? 원본 사진이 바뀌나요?

      네. 크롬·삼성인터넷·사파리 등 최신 모바일 브라우저에서 그대로 동작합니다. 원본 파일은 절대 수정되지 않으며, 압축본이 새 파일로 다운로드됩니다.
