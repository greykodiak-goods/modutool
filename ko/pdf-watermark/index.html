    ℹ️ 크기는 페이지 대각선 길이에 비례해 자동 계산됩니다. 한글 문구는 처음 적용할 때 한글 폰트(약 2.4MB)를 한 번 내려받습니다 — PDF 파일이 업로드되는 것이 아닙니다.

    사용 방법

    1. 위 영역에 PDF 파일 1개를 끌어다 놓거나, 클릭해서 선택합니다. 파일을 읽는 즉시 총 페이지 수가 목록에 표시됩니다.
    2. 워터마크 문구를 입력합니다. 기본값은 "대외비"이며 초안, 사본, 배포 금지 등 원하는 문구로 바꿀 수 있습니다. 문구가 비어 있으면 버튼이 비활성화됩니다.
    3. 배치(대각선·가로 중앙·하단), 색상(회색·빨강·파랑), 투명도(0.08~0.5), 크기 배율(0.5~2배)을 취향대로 조절합니다.
    4. 워터마크 넣기 버튼을 누르면 모든 페이지에 문구가 삽입되고, 원본명_워터마크.pdf 파일이 자동으로 다운로드됩니다. 원본 파일은 그대로 보존됩니다.

    이 도구가 다른 PDF 워터마크 사이트와 다른 점

    흔한 온라인 워터마크 서비스는 파일을 자기 회사 서버에 업로드한 뒤 처리해서 되돌려줍니다. 워터마크를 넣는 문서가 어떤 문서인지 생각해 보면 아이러니한 구조입니다 — 대외비 표시가 필요할 만큼 민감한 계약서·제안서·견적서를, 워터마크를 넣겠다고 남의 서버에 먼저 올리는 셈이니까요. 업로드된 파일이 언제 삭제되는지, 처리 과정에서 누가 열람할 수 있는지 사용자는 확인할 방법이 없습니다.

    ThisIsMyPDF의 PDF 워터마크는 파일을 어디로도 보내지 않습니다. 브라우저에 내장된 기술로 문구 삽입 작업 전체가 내 기기 안에서만 이뤄집니다. 그래서 세 가지 장점이 생깁니다.

    • 보안 — 파일이 기기 밖으로 나가지 않으므로 유출 가능성 자체가 없습니다.
    • 속도 — 업로드·다운로드 대기가 없어 수십 쪽짜리 문서도 몇 초 안에 끝납니다.
    • 무제한 — 서버 비용이 들지 않으니 파일 용량·횟수를 제한하지 않습니다.

    워터마크 옵션 자세히

    배치 — 대각선·가로 중앙·하단

    대각선(중앙)은 가장 널리 쓰이는 방식으로, 페이지 대각선 각도에 맞춰 문구를 비스듬히 눕혀 페이지 전체를 덮습니다. 캡처하거나 일부만 잘라내도 문구가 걸리기 때문에 무단 배포 억제 효과가 가장 큽니다. 가로(중앙)는 페이지 한가운데에 수평으로 배치해 상태 표시가 또렷하게 보이고, 하단은 본문을 거의 가리지 않아 사본 표시나 출처 표기처럼 은은하게 남기고 싶을 때 적합합니다.

    투명도 — 본문 가독성과의 균형

    투명도는 0.08(거의 안 보임)부터 0.5(뚜렷함)까지 조절할 수 있고, 기본값은 0.15입니다. 워터마크의 목적은 본문을 가리는 것이 아니라 문서의 상태를 알리는 것이므로, 인쇄·열람에 지장이 없는 옅은 값에서 시작해 필요할 때만 올리는 것을 권합니다.

    크기와 색상

    글자 크기는 페이지 대각선 길이에 비례해 자동 계산되므로 A4든 명함 크기든 페이지마다 알맞은 크기로 들어갑니다. 배율 슬라이더로 0.5~2배 사이에서 미세 조정할 수 있습니다. 색상은 서류 표준인 회색, 경고성 표시에 쓰는 빨강, 조직 안내 문구에 어울리는 파랑 세 가지를 제공합니다.

    💡 팁: 받는 사람이 본문을 읽어야 하는 문서(제안서·계약서 초안)라면 투명도를 0.12~0.2 사이로 낮춰 가독성을 지키세요. 반대로 "이 파일은 초안"임을 강하게 알려야 하는 내부 검토본은 투명도 0.3 이상 + 빨강 대각선이 효과적입니다.

    이럴 때 쓰세요

    • 외부에 보내는 계약서·견적서에 "대외비" 표시를 넣어 무단 전달을 억제할 때
    • 검토 중인 문서에 "초안" 표시를 넣어 확정본과 혼동되지 않게 할 때
    • 관공서·은행 제출용 서류 사본에 "사본" 표시를 넣어 원본 도용을 막을 때
    • 제안서·포트폴리오에 회사명이나 이름을 넣어 무단 배포·재사용을 억제할 때
    • 강의자료·전자문서에 배포 대상 표시(예: 2026년 1기 수강생용)를 넣을 때

    자주 묻는 질문

    파일이 정말 서버로 전송되지 않나요?

    네. 워터마크 삽입 전체가 브라우저 안에서, 내 컴퓨터(또는 휴대폰) 안에서만 처리됩니다. 한글 문구를 처음 적용할 때 이 사이트에서 한글 폰트 파일을 한 번 내려받는 통신이 있을 뿐, 여러분의 PDF가 밖으로 나가는 요청은 없습니다. 개발자도구의 네트워크 탭에서 직접 확인할 수 있습니다.

    넣은 워터마크를 나중에 지울 수 있나요?

    이 도구는 워터마크를 별도 주석이 아니라 페이지 내용에 직접 그려 넣기 때문에, 일반 뷰어에서는 지워지지 않습니다. 다만 전문 편집 도구를 쓰면 제거가 불가능하지는 않으므로, 워터마크는 완전한 복제 방지 장치가 아니라 무단 배포를 억제하고 문서 상태를 표시하는 용도로 이해하는 것이 좋습니다. 원본 파일은 변경되지 않으니 원본을 따로 보관하세요.

    한글 워터마크도 넣을 수 있나요?

    네. 대외비, 초안, 사본 같은 한글 문구를 넣으면 한글 글꼴(Noto Sans KR)이 자동으로 적용됩니다. 한글이 포함된 경우에만 폰트 파일을 한 번 내려받으며, 영문·숫자만 있는 문구는 폰트 다운로드 없이 즉시 처리됩니다.

    암호가 걸린 PDF에도 워터마크를 넣을 수 있나요?

    열람 암호(비밀번호)가 걸린 PDF는 보안 정책상 열 수 없어 안내 메시지가 표시됩니다. PDF 뷰어 등에서 암호를 해제해 저장한 파일로 다시 시도해 주세요.

    투명도와 크기는 어느 정도가 적당한가요?

    본문 가독성을 해치지 않으려면 기본값인 투명도 0.15 전후의 옅은 회색 대각선 배치를 권합니다. 문서 상태를 강하게 표시해야 하면 투명도를 0.3 이상으로 올리거나 빨강을 선택하세요. 크기는 페이지 대각선에 비례해 자동 계산되며, 배율 슬라이더로 0.5~2배 사이에서 조절할 수 있습니다.
