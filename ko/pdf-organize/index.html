PDF 페이지 삭제·순서 바꾸기🔒 서버 업로드 없음

미리보기를 보면서 페이지를 지우고, 옮기고, 회전합니다. 설치·가입 없이 무료.

🗂️ PDF 파일을 여기에 끌어다 놓거나 클릭해서 선택
페이지 미리보기가 나타나면 ✕ 삭제 · ◀ ▶ 이동 · ⟳ 회전

사용 방법

  1. PDF를 올리면 모든 페이지가 미리보기 그림으로 나타납니다.
  2. 필요 없는 페이지는 ✕를 눌러 삭제 표시(다시 누르면 복원), 순서를 바꿀 페이지는 ◀ ▶로 이동, 방향이 틀어진 페이지는 ⟳로 90도씩 회전합니다.
  3. 편집한 PDF 저장을 누르면 편집 결과가 새 파일로 다운로드됩니다. 원본은 그대로 남습니다.

왜 이 도구인가요?

PDF에서 한두 쪽을 지우는 간단한 일에도 대부분의 서비스는 번거로움을 요구합니다. 어떤 곳은 로그인을 해야 편집을 진행할 수 있고, 어떤 곳은 하루 사용 횟수를 제한하며, 설치형 프로그램은 광고를 함께 보여줍니다. 무엇보다 대부분 파일을 회사 서버에 업로드해서 처리합니다.

ThisIsMyPDF는 페이지 삭제·순서변경·회전 전 과정을 브라우저 안에서만 처리합니다. 파일이 기기 밖으로 나가지 않으니 계약서·등본·진단서 같은 민감한 문서도 안전하고, 업로드 대기가 없어 수십 쪽 문서도 몇 초면 끝납니다. 횟수·용량 제한도, 워터마크도 없습니다.

💡 팁: 스캔 앱으로 만든 PDF는 중간에 빈 쪽이나 흔들린 쪽이 끼기 쉽습니다. 여기서 지우고, 뒤집힌 쪽은 ⟳로 바로잡은 뒤, 여러 파일을 묶어야 하면 PDF 합치기로 마무리하세요.

이럴 때 쓰세요

  • 제출 서류에서 불필요한 쪽만 빼고 낼 때 (개인정보가 있는 쪽 제외 등)
  • 스캔 순서가 뒤죽박죽인 문서를 미리보기 보면서 재정렬할 때
  • 가로로 스캔돼 옆으로 누운 페이지만 회전시킬 때
  • 보고서에서 구버전 페이지를 교체하기 전 삭제할 때

함께 알아두면 좋은 것

이 도구는 페이지 단위 편집 전용입니다. 특정 페이지 범위를 새 파일로 추출하고 싶다면 PDF 나누기가 더 빠르고, 페이지를 이미지로 저장하려면 PDF → JPG 변환을 사용하세요. 열람 암호가 걸린 PDF는 보안 정책상 편집할 수 없으니 먼저 암호를 해제해야 합니다.

자주 묻는 질문

파일이 정말 서버로 전송되지 않나요?

네. 페이지 삭제·순서변경·회전 전 과정이 브라우저 안에서만 처리됩니다. 개발자도구 네트워크 탭을 열어 확인해도 파일 업로드 요청이 없습니다. 계약서나 등본 같은 민감한 문서도 안심하고 사용할 수 있습니다.

한 번에 여러 페이지를 지울 수 있나요?

네. 각 페이지 미리보기의 ✕ 버튼을 눌러 원하는 만큼 삭제 표시를 하고, 저장 버튼을 한 번만 누르면 됩니다. 잘못 눌렀다면 같은 자리의 복원 버튼으로 되돌릴 수 있습니다.

페이지 순서는 어떻게 바꾸나요?

각 페이지 카드의 ◀ ▶ 버튼으로 한 칸씩 이동합니다. 미리보기 그림을 보면서 옮기기 때문에 몇 쪽인지 외울 필요가 없습니다.

가로로 스캔된 페이지만 돌릴 수 있나요?

네. 회전은 페이지별로 적용됩니다. 돌리고 싶은 페이지의 ⟳ 버튼만 누르면 그 페이지만 90도씩 회전하며, 나머지 페이지는 그대로 유지됩니다.

원본 파일이 변경되나요?

아니요. 내 컴퓨터의 원본 PDF는 그대로 두고, 편집 결과가 새 파일로 다운로드됩니다. 결과가 마음에 안 들면 원본으로 다시 시도하면 됩니다.
