이미지 포맷 변환🔒 서버 업로드 없음

JPG·PNG·WEBP 상호 변환 — 화질 슬라이더와 투명 배경 처리까지 여러 장 한 번에. 설치·가입 없이 무료.

🔄 이미지 파일을 여기에 끌어다 놓거나 클릭해서 선택
JPG · PNG · WEBP · BMP · GIF, 여러 장 한 번에 가능

    ℹ️ JPG는 투명도가 없어, PNG·WEBP의 투명 영역은 흰색으로 채워집니다. 투명 배경을 유지하려면 출력 형식을 PNG나 WEBP로 선택하세요.

      사용 방법

      1. 위 영역에 이미지(JPG·PNG·WEBP·BMP·GIF)를 끌어다 놓거나, 클릭해서 여러 장을 한 번에 선택합니다.
      2. 출력 형식을 JPG·PNG·WEBP 중에서 고릅니다. 화질 슬라이더는 JPG·WEBP에서만 나타나며(85~95가 적당), PNG는 무손실이라 화질 설정이 필요 없습니다.
      3. 이미지 변환하기 버튼을 누르면 파일별로 처리되며, 목록에 원본 형식 → 결과 형식과 용량(예: PNG → JPG, 1.8 MB → 420 KB)이 표시됩니다.
      4. 파일별 ⬇ 버튼으로 하나씩 받거나 전체 다운로드로 한 번에 저장합니다. 변환 파일은 원본 이름에 새 확장자를 붙여 저장되며, 원본 파일은 그대로 남습니다.

      이 도구가 다른 변환 사이트와 다른 점

      대부분의 온라인 변환 사이트는 이미지를 자기 회사 서버에 업로드한 뒤 변환해서 되돌려줍니다. 업로드된 이미지가 언제 지워지는지, 처리 과정에서 누가 볼 수 있는지 사용자는 알 수 없습니다. 가족 사진, 신분증 사본, 계약서 스캔처럼 민감한 이미지라면 찜찜할 수밖에 없습니다.

      ThisIsMyPDF의 이미지 변환은 이미지를 어디로도 보내지 않습니다. 브라우저에 내장된 캔버스 기술로 변환 전체가 내 컴퓨터(또는 휴대폰) 안에서만 이뤄집니다. 그래서 세 가지 장점이 생깁니다.

      • 보안 — 이미지가 기기 밖으로 나가지 않으므로 유출 가능성 자체가 없습니다.
      • 속도 — 업로드·다운로드 대기가 없어 수십 장도 몇 초~수십 초면 끝납니다.
      • 무제한 — 서버 비용이 들지 않으니 장수·횟수·용량을 제한하지 않습니다.

      세로 사진도 올바르게 처리됩니다 — 스마트폰 카메라가 저장한 EXIF 회전 정보를 반영하므로, 변환 후에 사진이 옆으로 눕거나 뒤집히는 다른 도구의 고질적인 문제가 없습니다.

      어떤 형식을 골라야 하나요?

      각 형식은 잘하는 일이 다르며, 알맞은 형식을 고르는 것이 사실 변환하는 이유의 전부입니다.

      • 호환을 위한 WEBP → JPG. 요즘 점점 많은 사이트가 이미지를 WEBP로 저장하는데, 일부 메신저·구형 업로드 폼·데스크톱 프로그램은 여전히 WEBP를 열지 못합니다. WEBP를 JPG로 바꾸면 어디서나 열립니다 — 카톡으로 사진을 보낼 때, 민원 사이트에 업로드할 때, 구형 문서 편집기에 넣을 때 등, 이미지를 변환하는 가장 흔한 이유가 바로 이것입니다.
      • 투명 배경은 PNG로 유지. 회사 로고, 아이콘, 누끼 딴 서명은 배경이 비쳐 보여야 합니다. 이런 파일을 JPG로 바꾸면 투명 영역이 흰색으로 채워져 색이 있는 페이지에서 어색해지므로, 투명도를 살리려면 PNG(또는 WEBP)로 유지·변환하세요.
      • 용량 절감을 위한 WEBP 변환. 블로그·쇼핑몰·홈페이지용이라면 WEBP가 같은 화질에서 보통 JPG보다 25~35% 작은 파일을 만들어 로딩 속도와 검색 노출(SEO)에 유리합니다. 크롬·엣지·사파리·삼성인터넷 등 모든 최신 브라우저가 WEBP를 열 수 있어, 온라인에 올리는 이미지라면 안전한 선택입니다.
      • 무손실 사본은 PNG. 스크린샷·도표·글자가 또렷한 이미지는 PNG로 저장하면 가장자리에 압축 잡티 없이 선명하게 유지됩니다. JPG를 PNG로 바꿔도 JPG가 이미 버린 정보가 복원되지는 않지만, 이후 편집하고 다시 저장할 때 추가 손실을 막아줍니다.
      💡 팁: JPG·WEBP로 변환하면 대개 용량이 줄지만, 정확한 용량 제한을 맞춰야 한다면 변환 후 이미지 압축을 이어서 쓰세요. 픽셀 크기를 정확히 맞춰야 한다면 이미지 크기조절로 먼저 리사이즈하세요.

      HEIC(아이폰 사진) 안내

      최신 아이폰으로 찍은 사진은 애플의 고효율 형식인 HEIC로 저장되는 경우가 많습니다. 브라우저는 HEIC를 읽지 못하므로, 이 도구를 포함한 어떤 브라우저 도구도 HEIC를 직접 변환할 수 없습니다. 깨진 파일을 내주는 대신, 변환기는 HEIC 업로드를 감지해 건너뛴다고 분명히 표시합니다. 여기서 쓰려면 아이폰을 JPG로 촬영하도록 설정하거나(설정 → 카메라 → 포맷 → "높은 호환성"), 사진 앱에서 JPG로 내보내거나 공유한 뒤 사용하세요. 일단 JPG가 되면 다른 이미지와 똑같이 변환됩니다.

      이럴 때 쓰세요

      • WEBP 이미지가 필요한 곳에서 열리지 않거나 업로드가 안 될 때 JPG로 변환
      • 다른 형식에서 변환하거나 그대로 유지해야 할 투명 PNG 로고가 필요할 때
      • 더 빠르고 검색에 잘 노출되는 홈페이지를 위해 더 작은 WEBP 파일이 필요할 때
      • 양식이나 프로그램이 JPG·PNG만 받고 현재 형식을 거부할 때
      • 형식이 뒤섞인 이미지 묶음을 한 형식으로 통일할 때

      자주 묻는 질문

      파일이 정말 서버로 전송되지 않나요?

      네. 변환은 브라우저에 내장된 캔버스 기술로 내 기기 안에서만 처리됩니다. 개발자도구의 네트워크 탭을 열어 확인해도 파일 업로드 요청이 없습니다. 신분증 사본이나 가족 사진처럼 민감한 이미지도 안심하고 변환할 수 있습니다.

      어떤 형식끼리 변환할 수 있나요?

      JPG·PNG·WEBP·BMP·정지 GIF를 불러와 JPG·PNG·WEBP로 변환할 수 있습니다. 카톡·업로드 호환을 위한 WEBP→JPG, 용량 절감을 위한 PNG→WEBP, 무손실 사본이 필요한 JPG→PNG 같은 일상적인 경우를 모두 지원합니다. 움직이는 GIF는 첫 프레임만 변환됩니다.

      PNG를 JPG로 바꾸면 투명 배경은 어떻게 되나요?

      JPG는 투명도를 지원하지 않아, PNG나 WEBP를 JPG로 변환하면 투명 영역이 흰색으로 채워집니다. 로고나 아이콘처럼 투명 배경을 유지해야 한다면 출력 형식을 PNG나 WEBP로 선택하세요. 투명 픽셀이 채워진 경우 도구가 알려드립니다.

      아이폰 HEIC 사진도 변환되나요?

      직접 변환은 되지 않습니다. 브라우저가 애플의 HEIC/HEIF 형식을 읽지 못하기 때문에, 이 도구는 HEIC 파일을 감지해 깨진 결과 대신 건너뜁니다. 아이폰 설정 → 카메라 → 포맷에서 '높은 호환성'을 선택해 JPG로 촬영하거나, 사진 앱에서 JPG로 내보내거나 공유한 뒤 사용하면 문제없이 변환됩니다.

      JPG·PNG·WEBP 중 어떤 형식을 골라야 하나요?

      사진이나 카톡·구형 업로드 폼처럼 호환성이 가장 중요한 곳에는 JPG를 쓰세요. 투명 배경이 필요하거나 그림·스크린샷의 무손실 사본이 필요하면 PNG를 쓰세요. 같은 화질에서 파일을 가장 작게 만들고 싶은 웹 용도라면 WEBP가 좋습니다 — JPG보다 보통 25~35% 작고 투명도도 지원합니다.
