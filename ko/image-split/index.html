이미지 분할(인스타 격자)🔒 서버 업로드 없음

이미지 한 장을 세로 2분할, 인스타 캐러셀 3분할, 2×2, 3×3 격자로 — 원본 해상도 그대로 픽셀 단위로 잘라냅니다. 무료 — 설치·가입 없음.

🧩 이미지를 끌어다 놓거나 클릭해서 선택
한 번에 한 장 (JPG · PNG · WEBP · GIF · BMP)

점선이 잘리는 위치입니다. 조각 번호는 왼쪽 → 오른쪽, 위 → 아래 순입니다.

사용 방법

  1. 이미지를 위 영역에 끌어다 놓거나 클릭해서 선택합니다.
  2. 분할 방식을 고릅니다. 미리보기의 점선이 실제로 잘리는 위치입니다 — 3조각(세로 3분할)이 인스타그램 파노라마 캐러셀 형식, 3×3이 프로필 격자용입니다.
  3. 분할해서 다운로드를 누르면 각 조각이 원본 해상도 그대로 잘려 파일명_1, 파일명_2… 로 저장됩니다. 브라우저가 여러 파일 다운로드 허용을 물으면 허용을 누르세요.
  4. 업로드 순서: 캐러셀은 1 → 2 → 3 순서대로, 3×3 프로필 격자는 거꾸로(9번 먼저, 1번 마지막) — 인스타그램이 최신 게시물을 왼쪽 위에 놓기 때문입니다.

조각 번호는 이렇게 매겨집니다

조각은 항상 왼쪽에서 오른쪽, 위에서 아래 순으로 번호가 붙습니다: 3×3 격자라면 _1이 왼쪽 위, _9가 오른쪽 아래입니다. 번호가 예측 가능해야 업로드 순서도 정해집니다 — 옆으로 넘기는 캐러셀은 번호 순서대로, 프로필 격자는 역순으로 올리면 됩니다.

가로·세로가 딱 나누어떨어지지 않으면 정수 픽셀로 자르고 남는 픽셀은 마지막 열·행이 가져갑니다 — 1000px 3분할이면 333 + 333 + 334. 픽셀이 버려지거나 늘어나는 일은 없습니다.

원본 해상도 그대로, 리샘플링 없음

화면의 미리보기는 축소된 것이지만 잘라내기는 원본 픽셀에서 이루어집니다. 4000×3000 사진을 세로 3분할하면 약 1333×3000짜리 파일 3개가 나옵니다 — 축소되는 것이 없습니다. JPG 입력은 JPG(92% 품질)로, 그 외 형식은 무손실 PNG로 저장됩니다. 깨끗한 캔버스로 다시 저장되는 부수 효과로, GPS 위치 같은 EXIF 메타데이터는 조각에 옮겨가지 않습니다.

💡 팁: 인스타그램 캐러셀은 각 조각이 지원 비율(1:1, 4:5, 1.91:1)에 가까울 때 가장 잘 보입니다. 파노라마가 아주 길다면 이미지 자르기로 먼저 다듬어 각 조각이 4:5에 가깝게 만들어 보세요.

이런 때 좋아요

  • 인스타 파노라마 캐러셀 — 와이드 사진 한 장을 3장으로 넘겨 보기
  • 큰 그림 한 장으로 맞춰지는 3×3 프로필 격자 아트
  • 스캔한 페이지·포스터를 나눠서 인쇄
  • 좌우 짝이 맞아야 하는 채팅 배경·배너
  • 큰 그래픽 한 장을 포토월용 타일로 나누기

자주 묻는 질문

조각은 어떤 순서로 저장되고, 인스타 업로드 순서는 왜 중요한가요?

조각 번호는 왼쪽→오른쪽, 위→아래 순입니다 — _1이 왼쪽 위 조각입니다. 캐러셀(세로 3분할)은 1, 2, 3 번호 순서대로 추가하면 옆으로 넘길 때 자연스럽게 이어집니다. 3×3 프로필 격자는 반대입니다: 인스타그램은 최신 게시물을 왼쪽 위에 배치하므로 마지막 조각부터(9, 8, 7 … 1) 올려야 격자가 올바르게 맞춰집니다.

파일이 한 개만 다운로드됐어요. 나머지는 어디 있나요?

브라우저는 허용 전에는 한 페이지가 여러 파일을 연달아 저장하는 것을 막습니다. 이 도구는 다운로드 간격을 두고 저장하는데, 브라우저가 "여러 파일을 다운로드하려고 합니다" 같은 안내를 띄우면 허용을 누른 뒤 다시 분할하세요. 모든 조각이 번호 붙은 개별 파일로 저장됩니다.

분할하면 화질이 떨어지나요?

잘라내기 자체는 픽셀 단위로 정확합니다: 각 조각은 원본 해상도에서 그대로 오려내며 확대·축소·리샘플링이 없습니다. 다만 저장 시 다시 인코딩됩니다 — JPG 조각은 92% 품질로 저장되어 사진에서는 눈으로 구분되지 않고, PNG 조각은 무손실입니다. 조각을 전부 합치면 원본과 동일한 픽셀입니다.

이미지가 딱 나누어떨어지지 않으면 어떻게 되나요?

남는 픽셀은 마지막 열과 마지막 행이 가져갑니다. 예를 들어 가로 1000px을 3분할하면 333, 333, 334px 조각이 됩니다. 조각 간에 1~2px 차이가 날 수 있지만 눈으로는 보이지 않고, 픽셀이 버려지거나 늘어나는 일은 없습니다.

이미지가 서버로 업로드되나요?

아니요. 잘라내기는 브라우저 내장 캔버스 기능으로 전부 내 기기 안에서 처리됩니다. 브라우저 개발자도구의 네트워크 탭에서 파일이 전송되지 않는 것을 직접 확인할 수 있고, 탭을 닫으면 모든 것이 사라집니다.
