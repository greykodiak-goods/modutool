PDF 용량 줄이기🔒 서버 업로드 없음

PDF 파일을 3단계 화질 중 골라 압축해 용량을 줄입니다. 설치·가입·용량제한 없이 무료.

🗜️ PDF 파일을 여기에 끌어다 놓거나 클릭해서 선택
쪽수와 용량을 확인한 뒤 압축 강도를 고르세요
    ⚠️ 압축 시 각 페이지가 이미지로 재구성되어 결과 PDF에서는 텍스트 선택·검색·복사가 되지 않습니다. 스캔 문서·이메일 첨부·용량 제한 제출에 알맞고, 텍스트 보존이 필요한 문서에는 원본을 따로 보관하세요.

    사용 방법

    1. 위 영역에 압축할 PDF 파일을 끌어다 놓거나, 클릭해서 선택합니다. 파일을 올리면 전체 쪽수와 현재 용량이 바로 표시됩니다.
    2. 압축 강도를 고릅니다. 대부분은 기본값인 권장이면 충분하고, 제출 용량 제한이 빡빡하면 강한 압축, 도면·작은 글씨가 중요하면 최고 화질을 고르세요.
    3. PDF 압축하기 버튼을 누르면 페이지를 하나씩 다시 그려 압축하고, 완료되면 원본이름_압축.pdf 파일이 자동으로 다운로드됩니다.
    4. 결과 화면에서 원본 용량 → 압축 후 용량과 절감률을 확인합니다. 마음에 들지 않으면 강도를 바꿔 다시 누르면 됩니다.

    이 도구가 다른 PDF 압축 사이트와 다른 점

    흔히 쓰는 온라인 PDF 압축 서비스는 파일을 자기 회사 서버에 업로드한 뒤 압축해서 되돌려줍니다. 업로드된 문서가 언제 삭제되는지, 처리 과정에서 누가 볼 수 있는지 사용자는 확인할 방법이 없습니다. 계약서, 등본, 진단서, 급여명세서처럼 민감한 문서라면 특히 불안할 수밖에 없습니다.

    ThisIsMyPDF의 PDF 용량 줄이기는 파일을 어디로도 보내지 않습니다. 브라우저에 내장된 PDF 렌더링 기술로 각 페이지를 화면에 그리듯 그린 뒤, 그 결과를 압축 이미지로 담아 새 PDF를 만드는 방식이라 전 과정이 내 컴퓨터(또는 휴대폰) 안에서만 이뤄집니다. 그래서 보안(파일이 기기 밖으로 나가지 않음)·속도(업로드 대기 없음)·무제한(쪽수·횟수·용량 제한 없음)이라는 세 가지 장점이 생깁니다.

    정직하게 알려드립니다 — 압축의 원리와 한계

    이 도구는 각 페이지를 정해진 해상도로 다시 그린 뒤 JPEG 이미지로 담아 새 PDF를 만듭니다. 스캔본처럼 페이지가 원래 큰 이미지인 문서에서 용량이 극적으로 줄어드는 이유입니다. 대신 분명한 대가가 있습니다. 압축된 PDF의 페이지는 이미지이므로 텍스트 선택·검색·복사가 되지 않고, 화면 낭독기 같은 접근성 도구도 내용을 읽지 못합니다.

    • 적합 — 스캔한 계약서·증빙, 이메일 첨부, 관공서·입사 지원 시스템의 용량 제한 제출, 보관용 사본
    • 부적합 — 나중에 내용을 검색·복사·인용해야 하는 보고서·논문·전자책 (이 경우 원본을 반드시 따로 보관하세요)

    압축 강도는 이렇게 골라요

    • 강한 압축 — 용량이 가장 작아집니다. 화면에서 읽는 데는 문제없지만 확대하면 글자 가장자리가 다소 뭉개질 수 있습니다. 5MB·10MB처럼 빡빡한 업로드 제한에 맞출 때 고르세요.
    • 권장 (기본값) — 화질과 용량의 균형점입니다. 일반 문서·스캔본 제출, 이메일 첨부 등 대부분의 용도에 알맞습니다.
    • 최고 화질 — 원본에 가까운 선명함을 유지하는 대신 용량 감소 폭이 작습니다. 도면, 작은 글씨의 표, 인쇄할 문서에 권하세요.

    압축이 끝나면 원본과 결과 용량, 절감률이 바로 표시되므로 강도를 바꿔가며 여러 번 눌러 비교해도 됩니다. 텍스트 위주라 원래 가벼운 PDF나 이미 강하게 압축된 파일은 이미지로 다시 그리면 오히려 커질 수 있는데, 이때는 파일을 다운로드하지 않고 원본을 그대로 쓰시라고 안내해 드립니다.

    이럴 때 쓰세요

    • 스캔한 PDF가 커서 이메일 첨부 제한(보통 10~25MB)에 걸릴 때
    • 관공서 민원·정부 지원사업·입사 지원 사이트의 업로드 용량 제한(5~10MB)에 맞춰야 할 때
    • 휴대폰으로 촬영해 만든 스캔 계약서·증빙 서류가 수십 MB로 부풀었을 때
    • 수업자료·브로슈어를 메신저·게시판에 가볍게 공유하고 싶을 때
    💡 팁: 압축 전에 PDF 페이지 정리로 불필요한 페이지를 지우면 용량이 한 번 더 줄어듭니다. 사진(JPG) 상태라면 이미지 압축으로 먼저 줄인 뒤 이미지 → PDF 변환을 거치는 편이 화질 손실이 더 적습니다.

    자주 묻는 질문

    파일이 정말 서버로 전송되지 않나요?

    네. 압축은 브라우저에 내장된 기술로 내 기기 안에서만 처리됩니다. 개발자도구의 네트워크 탭을 열어 확인해도 파일 업로드 요청이 없습니다. 계약서·등본·진단서 같은 민감한 문서도 안심하고 사용할 수 있습니다.

    압축하면 텍스트 선택·검색이 안 된다던데 사실인가요?

    사실입니다. 이 도구는 각 페이지를 이미지로 다시 그려 용량을 줄이는 방식이라, 압축된 PDF에서는 텍스트 선택·검색·복사가 되지 않습니다. 원래부터 이미지인 스캔 문서, 이메일 첨부, 용량 제한 제출용으로는 최적이지만, 나중에 내용을 검색하거나 복사해야 하는 문서라면 원본을 따로 보관하세요.

    압축 강도는 어떤 것을 고르면 되나요?

    대부분은 기본값인 권장이면 화면·인쇄 모두 무난합니다. 용량 제한이 빡빡한 제출(예: 5MB·10MB 제한)에는 강한 압축을, 도면·작은 글씨처럼 선명함이 중요한 문서에는 최고 화질을 고르세요. 압축 후 결과 용량이 바로 표시되므로 강도를 바꿔가며 다시 눌러 비교해도 됩니다.

    얼마나 줄어드나요? 오히려 커질 수도 있나요?

    스캔본·사진이 많은 PDF는 보통 절반 이하로 줄고, 강한 압축에서는 70~90%까지 줄기도 합니다. 반면 텍스트 위주라 원래 가벼운 PDF나 이미 강하게 압축된 파일은 이미지로 다시 그리면 오히려 커질 수 있습니다. 이 경우 도구가 다운로드하지 않고 원본을 그대로 쓰시라고 안내합니다.

    암호가 걸린 PDF도 압축되나요?

    열람 암호가 걸린 PDF는 내용을 읽을 수 없어 압축되지 않고 안내 메시지가 표시됩니다. 먼저 암호를 해제한 파일로 다시 시도해 주세요.
