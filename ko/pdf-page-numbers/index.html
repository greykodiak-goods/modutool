PDF 쪽번호 넣기🔒 서버 업로드 없음

전체 페이지에 쪽번호(페이지 번호)를 한 번에 삽입합니다. 위치·형식·시작 번호 선택, 설치·가입·용량제한 없이 무료.

🔢 쪽번호를 넣을 PDF를 여기에 끌어다 놓거나 클릭해서 선택
파일 1개 — 선택하면 총 페이지 수가 바로 표시됩니다

    사용 방법

    1. 위 영역에 쪽번호를 넣을 PDF 파일 1개를 끌어다 놓거나, 클릭해서 선택합니다. 파일을 읽는 즉시 총 페이지 수가 목록에 표시됩니다.
    2. 위치(상단·하단 × 왼쪽·가운데·오른쪽 6곳)와 형식(1 / 1 / 총N / - 1 - / 1페이지)을 고릅니다. 필요하면 시작 번호와 글자 크기를 바꾸고, 표지가 있는 문서라면 첫 페이지 제외를 켭니다.
    3. 쪽번호 넣기 버튼을 누르면 전체 페이지에 번호가 삽입되고, 원본명_쪽번호.pdf 파일이 자동으로 다운로드됩니다.
    4. 완료 문구에 몇 쪽에 번호가 들어갔는지 표시되니, 첫 페이지 제외 여부까지 바로 확인할 수 있습니다.

    이 도구가 다른 쪽번호 사이트와 다른 점

    흔한 온라인 PDF 페이지 번호 서비스는 파일을 자기 회사 서버에 업로드한 뒤 번호를 넣어 되돌려줍니다. 업로드된 문서가 언제 삭제되는지, 처리 과정에서 누가 열람할 수 있는지 사용자는 확인할 방법이 없습니다. 회사 보고서, 계약서, 논문 초안처럼 외부에 나가면 곤란한 문서라면 불안할 수밖에 없습니다.

    ThisIsMyPDF의 쪽번호 넣기는 파일을 어디로도 보내지 않습니다. 번호 계산부터 글자 삽입, 저장까지 전 과정이 브라우저에 내장된 기술로 내 기기 안에서만 이뤄집니다. 그래서 세 가지 장점이 생깁니다.

    • 보안 — 파일이 기기 밖으로 나가지 않으므로 유출 가능성 자체가 없습니다.
    • 속도 — 업로드·다운로드 대기가 없어 수백 쪽짜리 문서도 몇 초 안에 끝납니다.
    • 무제한 — 서버 비용이 들지 않으니 파일 용량·횟수를 제한하지 않습니다.

    형식·위치·시작 번호 고르기

    형식은 네 가지입니다. 1은 숫자만 깔끔하게, - 1 -은 인쇄물에서 흔한 장식형, 1페이지는 한글 표기형입니다. 1 / 총N은 "3 / 12"처럼 전체 쪽수를 함께 보여주는 형식인데, 여기서 총N은 쪽번호가 실제로 들어가는 페이지 수입니다. 즉 10쪽짜리 문서에서 첫 페이지 제외를 켜면 표지를 뺀 9쪽에만 번호가 들어가고, 표기도 1 / 9 ~ 9 / 9로 자동으로 맞춰집니다. 표지까지 세어 1 / 10로 어긋나는 일이 없습니다.

    위치는 하단 가운데가 기본이고, 상단·하단 × 왼쪽·가운데·오른쪽 여섯 곳 중에서 고를 수 있습니다. 여백은 가장자리에서 약 1cm(28pt)로, 일반적인 문서 여백 안에 안전하게 들어갑니다. 본문이나 기존 번호와 겹치면 다른 구석으로 바꿔 다시 만들면 됩니다 — 원본은 그대로 두고 새 파일로 저장되니 몇 번이든 부담이 없습니다.

    시작 번호는 기본 1이지만 자유롭게 바꿀 수 있습니다. 분책된 자료를 이어서 번호 매길 때(1권이 48쪽에서 끝났으면 2권은 49부터), 또는 표지·목차 뒤 본문부터 특정 번호로 시작해야 하는 제출 규정에 맞출 때 유용합니다. 1페이지 형식을 고르면 한글 표시용 글꼴이 그때만 자동으로 불러와지며, 숫자 형식은 추가 다운로드 없이 즉시 처리됩니다.

    💡 팁: 여러 파일을 합친 뒤 통합 쪽번호를 매기고 싶다면 PDF 합치기로 먼저 하나로 묶고 여기서 번호를 넣으세요. 페이지 순서를 바꾸거나 필요 없는 쪽을 지운 뒤 번호를 매기려면 PDF 페이지 정리를 먼저 거치면 됩니다. 번호는 항상 마지막 단계에 넣는 것이 다시 만들 일이 없어 편합니다.

    이럴 때 쓰세요

    • 보고서·논문·과제 제출 규정에 "페이지 번호 필수"가 있을 때 — 학교·학회·관공서 양식은 대부분 하단 가운데 번호를 요구합니다
    • 여러 장 스캔한 계약서·첨부서류에 페이지 표기를 넣어 "몇 쪽 중 몇 쪽"인지 명확히 할 때 (1 / 총N 형식)
    • 인쇄·제본 전 낱장이 섞여도 순서를 되찾을 수 있게 번호를 박아둘 때
    • 표지·간지가 있는 문서에서 표지는 건너뛰고 본문부터 번호를 시작할 때 (첫 페이지 제외)
    • 분책된 강의자료·매뉴얼을 권마다 이어지는 번호로 맞출 때 (시작 번호 조정)

    자주 묻는 질문

    파일이 정말 서버로 전송되지 않나요?

    네. 쪽번호 삽입 작업 전체가 브라우저에 내장된 기술로 내 컴퓨터(또는 휴대폰) 안에서만 처리됩니다. 한글 형식(1페이지)을 고르면 표시용 한글 글꼴 파일 하나를 저희 사이트에서 받아올 뿐, 사용자의 PDF는 어떤 경우에도 기기 밖으로 나가지 않습니다.

    "1 / 총N" 형식의 총N은 어떻게 정해지나요?

    쪽번호가 실제로 들어가는 페이지 수입니다. 예를 들어 10쪽짜리 문서에서 첫 페이지 제외를 켜면 2쪽부터 번호가 들어가고, 표기는 1 / 9부터 9 / 9까지가 됩니다. 표지를 뺀 본문 기준으로 총수가 맞춰지는 방식입니다.

    시작 번호를 1이 아닌 숫자로 바꿀 수 있나요?

    네. 시작 번호 칸에 원하는 숫자를 넣으면 첫 번호가 그 값부터 시작합니다. 여러 파일로 나뉜 문서를 이어서 번호 매길 때(예: 1권이 48쪽에서 끝났다면 2권은 49부터) 유용합니다.

    암호가 걸린 PDF에도 쪽번호를 넣을 수 있나요?

    열람 암호(비밀번호)가 걸린 PDF는 보안 정책상 열 수 없어 안내 메시지가 표시됩니다. PDF 뷰어 등에서 암호를 해제해 저장한 파일로 다시 시도해 주세요.

    기존 내용이나 이미 있는 쪽번호와 겹치면 어떻게 하나요?

    쪽번호는 페이지 위에 덧그려지는 방식이라 같은 자리에 글자나 기존 번호가 있으면 겹쳐 보일 수 있습니다. 위치를 6곳(상단·하단 × 왼쪽·가운데·오른쪽) 중에서 바꿔 비어 있는 구석을 고르면 대부분 해결됩니다. 원본 파일은 그대로 남고 새 파일로 저장되니 부담 없이 다시 시도할 수 있습니다.
