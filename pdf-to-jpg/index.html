    How this tool differs from other PDF-to-image sites

    Most popular online PDF to JPG services upload your file to their own servers, convert it there, and send the result back. You have no way of knowing when the uploaded document gets deleted or who can see it during processing. That is an uncomfortable thought when the file is a contract, an ID scan, a medical report, or a pay stub.

    ThisIsMyPDF's PDF to JPG converter never sends your file anywhere. It uses the PDF rendering technology built into your browser to draw each page — just as if displaying it on screen — and saves the result directly as a JPG, so the entire conversion happens on your computer (or phone). That design brings three advantages.

    • Privacy — your file never leaves the device, so there is simply nothing to leak.
    • Speed — no upload/download queue means even documents dozens of pages long finish quickly.
    • No limits — with no server costs, there are no caps on pages, uses, or file size, and it stays free.

    How do I choose the quality (scale)?

    The scale determines the resolution at which each page is rendered relative to its original size. PDFs measure an inch as 72 points, so image size in pixels = page size (pt) × scale. Using the most common Letter/A4-class page (A4 is 595×842 pt) as a reference:

    • Standard (1.5x) — about 893×1263 px (roughly 108 DPI). Light files, plenty for screen sharing, chat, and previews.
    • High (2x) — about 1190×1684 px (roughly 144 DPI). The sensible default for submitting documents or posting to a blog.
    • Maximum (3x) — about 1786×2526 px (roughly 216 DPI). Recommended for drawings, fine-print contract clauses, and anything headed to a printer.

    Higher scales give sharper images at the cost of larger files and longer conversion times. On memory-constrained devices like phones, start with Standard (1.5x) when converting hundreds of pages. JPG quality is fixed at 92%, so the scale is the only choice you need to make. For very small pages such as business cards or banners, the tool automatically ensures the long edge is at least 1200 px regardless of scale, so every document stays sharp enough to zoom into.

    Frequently asked questions

    Are my files really never sent to a server?

    Yes. The conversion runs entirely on your device using technology built into your browser. If you open the Network tab in your browser's developer tools, you will see no file upload requests. Even sensitive documents like contracts or ID scans are safe to use.

    How are multi-page PDFs saved?

    Each page is saved as one JPG file and downloaded automatically in order, named filename_1.jpg, filename_2.jpg, and so on. If your browser asks whether to allow multiple file downloads, click Allow. Once allowed, it will not ask again.

    Which quality setting should I choose?

    Use Standard (1.5x) for screen sharing and previews, High (2x) for submitting or publishing documents, and Maximum (3x) for drawings and print material where you need to zoom in on fine text. Higher settings produce sharper images, but files get larger and conversion takes longer.

    Can password-protected PDFs be converted?

    A PDF locked with an open password cannot be read, so it is not converted and a notice is shown instead. Please remove the password first and try again with the unlocked file.

    Does it work on my phone?

    Yes. Modern mobile browsers such as Chrome, Samsung Internet, and Safari are supported. However, converting a PDF with hundreds of pages at Maximum quality can be slow depending on your device's memory, so we recommend Standard or High quality on mobile.
