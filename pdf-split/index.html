PDF 나누기🔒 서버 업로드 없음

원하는 페이지만 추출하거나 페이지마다 개별 PDF로 분할합니다. 설치·가입·용량제한 없이 무료.

✂️ 나눌 PDF 파일을 여기에 끌어다 놓거나 클릭해서 선택
파일 1개 — 선택하면 총 페이지 수가 바로 표시됩니다

    ℹ️ 페이지 수만큼 PDF가 0.4초 간격으로 순차 다운로드됩니다. 브라우저가 "여러 파일 다운로드 허용" 여부를 물으면 허용을 눌러 주세요.

    사용 방법

    1. 위 영역에 나눌 PDF 파일 1개를 끌어다 놓거나, 클릭해서 선택합니다. 파일을 읽는 즉시 총 페이지 수가 목록에 표시됩니다.
    2. 모드를 고릅니다. 필요한 페이지만 뽑아 한 파일로 만들려면 범위 추출, 모든 페이지를 낱장 PDF로 만들려면 전체 분할을 선택하세요.
    3. 범위 추출을 골랐다면 1-3,5,7-9처럼 쉼표와 하이픈으로 페이지를 지정합니다.
    4. 버튼을 누르면 즉시 처리되어 자동으로 다운로드됩니다. 파일명은 원본명_1-3.pdf(범위 추출), 원본명_p1.pdf(전체 분할) 형식이라 어떤 페이지인지 바로 알 수 있습니다.

    이 도구가 다른 PDF 나누기 사이트와 다른 점

    흔한 온라인 PDF 분할 서비스는 파일을 자기 회사 서버에 업로드한 뒤 나눠서 되돌려줍니다. 업로드된 문서가 언제 삭제되는지, 처리 과정에서 누가 열람할 수 있는지 사용자는 확인할 방법이 없습니다. 계약서, 주민등록등본, 진단서, 급여명세서처럼 개인정보가 담긴 문서라면 더욱 불안할 수밖에 없습니다.

    모두의툴의 PDF 나누기는 파일을 어디로도 보내지 않습니다. 브라우저에 내장된 기술로 페이지 추출·분할 작업 전체가 내 기기 안에서만 이뤄집니다. 그래서 세 가지 장점이 생깁니다.

    • 보안 — 파일이 기기 밖으로 나가지 않으므로 유출 가능성 자체가 없습니다.
    • 속도 — 업로드·다운로드 대기가 없어 수십 MB 문서도 몇 초 안에 끝납니다.
    • 무제한 — 서버 비용이 들지 않으니 파일 용량·횟수를 제한하지 않습니다.

    두 가지 분할 모드

    ① 범위 추출 — 필요한 페이지만 한 파일로

    페이지 번호를 쉼표로 구분해 입력하면 해당 페이지만 모은 새 PDF 한 개가 만들어집니다. 연속 구간은 하이픈으로 줄여 쓸 수 있습니다.

    • 5 — 5쪽 한 장만 추출
    • 2-4 — 2쪽부터 4쪽까지 연속 추출
    • 1-3,5,7-9 — 1~3쪽 + 5쪽 + 7~9쪽, 총 7쪽을 입력한 순서대로 배치

    입력한 범위가 실제 페이지 수를 벗어나거나 형식이 틀리면 어떤 부분이 문제인지 바로 알려주므로, 페이지 수가 많은 문서도 실수 없이 뽑을 수 있습니다.

    ② 전체 분할 — 페이지마다 개별 PDF로

    문서의 모든 페이지가 각각 독립된 PDF 파일이 됩니다. 10쪽짜리 문서라면 원본명_p1.pdf부터 원본명_p10.pdf까지 10개 파일이 0.4초 간격으로 차례대로 내려받아집니다. 브라우저는 여러 파일 연속 다운로드를 기본으로 막아두는 경우가 많으니, 첫 파일 직후 뜨는 "여러 파일 다운로드 허용" 알림에서 허용을 눌러 주세요.

    💡 팁: 나눈 페이지를 다른 순서로 다시 묶고 싶다면 PDF 합치기를 이어서 사용하세요. 예를 들어 부록을 맨 앞으로 옮기고 싶을 때, 여기서 부록과 본문을 각각 추출한 뒤 합치기에서 순서만 바꾸면 재편집 프로그램 없이 끝납니다.

    이럴 때 쓰세요

    • 수십 쪽짜리 계약서에서 서명 페이지나 특약 부분만 뽑아 상대방·기관에 제출할 때
    • 한 번에 스캔한 묶음 파일에서 등본·통장사본·신분증 사본을 문서별로 분리해 보관할 때
    • 강의자료·전자책을 챕터 단위로 잘라 필요한 부분만 공유하거나 태블릿에 넣을 때
    • 이메일·팩스·업로드 용량 제한에 걸린 대용량 PDF를 몇 개의 작은 파일로 나눌 때
    • 발표자료에서 배포용 요약본(표지+핵심 페이지)만 추려 만들 때

    자주 묻는 질문

    파일이 정말 서버로 전송되지 않나요?

    네. 분할 작업 전체가 브라우저에 내장된 기술로 내 컴퓨터(또는 휴대폰) 안에서만 처리됩니다. 개발자도구의 네트워크 탭을 열어 확인해도 파일 업로드 요청이 없습니다. 계약서·등본·진단서 같은 민감한 문서도 안심하고 사용할 수 있습니다.

    페이지 범위는 어떻게 입력하나요?

    쉼표로 구분하고 연속 구간은 하이픈으로 적습니다. 예를 들어 1-3,5,7-9를 입력하면 1~3쪽, 5쪽, 7~9쪽 — 총 7쪽이 입력한 순서대로 새 PDF 한 개로 만들어집니다. 한 쪽만 뽑으려면 숫자 하나(예: 5)만 적으면 됩니다.

    전체 분할을 눌렀는데 파일이 하나만 받아져요. 왜 그런가요?

    브라우저가 여러 파일 연속 다운로드를 기본으로 차단하기 때문입니다. 첫 파일이 받아진 직후 주소창 근처에 뜨는 "여러 파일 다운로드 허용" 알림에서 허용을 누른 뒤 다시 실행하면 모든 페이지가 순서대로 내려받아집니다.

    암호가 걸린 PDF도 나눌 수 있나요?

    열람 암호(비밀번호)가 걸린 PDF는 보안 정책상 열 수 없어 안내 메시지가 표시됩니다. PDF 뷰어 등에서 암호를 해제해 저장한 파일로 다시 시도해 주세요.

    나눈 PDF의 화질이나 내용이 달라지나요?

    아니요. 페이지를 다시 렌더링하지 않고 원본 페이지 데이터를 그대로 복사하는 방식이라 글자·이미지 화질, 글꼴, 링크가 원본과 동일하게 유지됩니다.
